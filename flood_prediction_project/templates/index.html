How It Works

  1. Enter weather parameters — temperature, humidity, cloud cover, and rainfall figures for the region.
  2. Data is preprocessed and scaled using the same pipeline applied during training.
  3. The trained model generates a prediction — flood or no flood — with an estimated probability.
  4. Results are logged to the prediction history for later review and analysis.
